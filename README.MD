Proje Amacı
Dünya genelindeki siber güvenlik yarışmalarını (CTF) ve konferansları (BlackHat, DEFCON) takip eden, kullanıcının ilgi alanına göre filtreleyen bir platform.

Beklenen Özellikler
Etkinlik Takvimi: Yaklaşan CTF ve konferansları takvim üzerinde gösterme.
Otomatik Veri Çekme: CTFtime ve konferans web sitelerinden veri kazıma (scraping).
Kişiselleştirme: Kullanıcının ilgilendiği kategorilere (Web, Crypto, Pwn) göre bildirim gönderme.
Arşiv: Geçmiş etkinliklerin write-up ve sunum dosyalarına erişim sağlama.
Teknik Odak
Yöntem: Web Scraping, Calendar Integration
